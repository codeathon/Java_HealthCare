Patient Health Care Application Software in Java:

This is an all inclusive Patient Health care application to be used
by Professional medical practitioners to maintain information about their 
patient history.

Feature: Patient Family is linked if they have accounts at the same hospital.

This Application software is built entirely in Java and uses all the 
Object Oriented Features of Java like
Inheritance
Interfaces, Ploymorphism
Encapsulation etc.

It uses some  of the advanced features in Java 2EE like 
EJB ( Enterprise Java Beans)
EntityManager
JPA ( Java Persistence API)
JNDI (Java Naming and Directory Interface)

Design is fairly simple and clean.
2 Entity Beans:
   PatientBean
   DoctorBean
